# AniStock

## 1.这是什么？
将二次元人物变成股票，加上股票生成的算法，模拟真实世界的股票（当然并不像）  
目前还是有很大的随机性，往后的开发中会调整生成算法，尽量靠近真实
## 2.开发技术
1. Django
2. MySQL/Maraidb
